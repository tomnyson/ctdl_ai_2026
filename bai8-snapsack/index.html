Knapsack Visualizer 0/1 DP

Minh hoạ thuật toán cái túi bằng bảng quy hoạch động. Có thể chạy từng bước, chạy tự động, lùi bước, chỉnh tốc độ và xem các ô được dùng trong công thức.

1. Dữ liệu đầu vào

Mỗi dòng: Tên, Trọng lượng, Giá trị. Ví dụ: A,2,3. Bài toán dùng dạng 0/1: mỗi vật phẩm chỉ được lấy tối đa 1 lần.

2. Vật phẩm

Tốc độ chạy 700 ms/bước
Bước hiện tại 0 / 0
Ô đang xét —
Giá trị tối ưu 0
Trọng lượng lấy 0

Giải thích bước

Nhấn “Tạo bảng”, sau đó dùng “Từng bước” hoặc “Chạy”.

dp[i][w] = max(dp[i-1][w], dp[i-1][w-weight[i]] + value[i])

Chú thích màu

Ô đang tính dp[i][w]
Không lấy vật phẩm: dp[i-1][w]
Lấy vật phẩm: dp[i-1][w-weight] + value
Ô kết quả cuối cùng
Tiến độ 0%

Code Python tương ứng

Đoạn code dưới đây là bản Python của thuật toán 0/1 Knapsack đang được visualizer mô phỏng. 

def knapsack_01(items, capacity):
    """
    items: list tuple dạng (name, weight, value)
    capacity: sức chứa tối đa của balo

    Trả về:
    - best_value: giá trị tối ưu
    - picked_items: danh sách vật phẩm được chọn
    - dp: bảng quy hoạch động
    """
    n = len(items)

    # dp[i][w] = giá trị lớn nhất khi xét i vật phẩm đầu tiên
    # với sức chứa hiện tại là w
    dp = [[0] * (capacity + 1) for _ in range(n + 1)]

    # Tạo bảng DP
    for i in range(1, n + 1):
        name, weight, value = items[i - 1]

        for w in range(capacity + 1):
            # Trường hợp 1: không lấy vật phẩm i
            skip = dp[i - 1][w]

            # Trường hợp 2: lấy vật phẩm i nếu đủ sức chứa
            take = float("-inf")
            if weight <= w:
                take = dp[i - 1][w - weight] + value

            dp[i][w] = max(skip, take)

    # Truy vết để tìm vật phẩm được chọn
    picked_items = []
    w = capacity

    for i in range(n, 0, -1):
        if dp[i][w] != dp[i - 1][w]:
            name, weight, value = items[i - 1]
            picked_items.append((name, weight, value))
            w -= weight

    picked_items.reverse()
    best_value = dp[n][capacity]

    return best_value, picked_items, dp


def print_dp_table(items, capacity, dp):
    """In bảng DP ra terminal."""
    header = ["i/w"] + list(range(capacity + 1))
    print("\t".join(map(str, header)))

    for i, row in enumerate(dp):
        if i == 0:
            label = "0"
        else:
            label = items[i - 1][0]

        print(label, end="\t")
        print("\t".join(map(str, row)))


if __name__ == "__main__":
    items = [
        ("A", 2, 3),
        ("B", 3, 4),
        ("C", 4, 5),
        ("D", 5, 8),
        ("E", 9, 10),
    ]

    capacity = 10

    best_value, picked_items, dp = knapsack_01(items, capacity)

    print("Gia tri toi uu:", best_value)
    print("Vat pham duoc chon:")
    for name, weight, value in picked_items:
        print(f"- {name}: weight={weight}, value={value}")

    print("\nBang DP:")
    print_dp_table(items, capacity, dp)
