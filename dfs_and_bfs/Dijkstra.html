Ghi chú cho sinh viên:

  • Các số nằm nhỏ bên trên mỗi vòng tròn (node) là khoảng cách ngắn nhất hiện tại tính từ điểm A. Ban đầu tất cả đều là Vô cực (∞), riêng A là 0.
  • Dijkstra không quan tâm bạn đi qua bao nhiêu node, nó chỉ quan tâm tổng trọng số (cost) nhỏ nhất. Trong ví dụ, đi từ A->B->E qua 3 node tốn (4+3=7), trong khi đi từ A->D->E cũng 3 node nhưng chỉ tốn (1+2=3).
  • Nút Random trọng số giúp bạn tạo ra các bài toán mới để tự luyện tập.

Mã nguồn Python - Thuật toán Dijkstra

Đoạn code sử dụng heapq (Priority Queue) để tối ưu thời gian tìm kiếm Node có khoảng cách nhỏ nhất, giúp thuật toán chạy nhanh hơn.

import heapq

def dijkstra(graph, start, target):
    # Khởi tạo khoảng cách ban đầu là vô cực, riêng điểm bắt đầu là 0
    distances = {node: float('infinity') for node in graph}
    distances[start] = 0
    
    # Lưu lại dấu vết để truy ngược đường đi
    previous_nodes = {node: None for node in graph}
    
    # Hàng đợi ưu tiên (Priority Queue) lưu Tuple: (khoảng_cách, tên_node)
    pq = [(0, start)]
    
    while pq:
        current_distance, current_node = heapq.heappop(pq)
        
        # Nếu đã đến đích thì dừng thuật toán sớm
        if current_node == target:
            break
            
        # Bỏ qua nếu tìm thấy một khoảng cách lớn hơn khoảng cách hiện tại đã lưu
        if current_distance > distances[current_node]:
            continue
            
        # Duyệt qua các node kề
        for neighbor, weight in graph[current_node].items():
            distance = current_distance + weight
            
            # Cập nhật nếu tìm được đường đi ngắn hơn
            if distance < distances[neighbor]:
                distances[neighbor] = distance
                previous_nodes[neighbor] = current_node
                heapq.heappush(pq, (distance, neighbor))
                
    # Truy vết đường đi ngắn nhất
    path = []
    current = target
    while current is not None:
        path.insert(0, current)
        current = previous_nodes[current]
        
    return distances[target], path

# --- THỰC THI VỚI ĐỒ THỊ GIỐNG TRONG MÔ PHỎNG ---
graph = {
    'A': {'D': 1, 'B': 4},
    'B': {'A': 4, 'C': 2, 'E': 3},
    'C': {'B': 2, 'F': 5},
    'D': {'A': 1, 'E': 2},
    'E': {'B': 3, 'D': 2, 'F': 1},
    'F': {'C': 5, 'E': 1}
}

cost, path = dijkstra(graph, 'A', 'E')
print(f"Chi phí thấp nhất: {cost}")
print(f"Đường đi: {' -> '.join(path)}")
