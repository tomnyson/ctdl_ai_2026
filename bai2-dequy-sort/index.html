Điều hướng bài tập đệ quy và sắp xếp

Trang trung tâm để mở nhanh các bài mô phỏng thuật toán trong thư mục hiện tại. Chọn một chủ đề để vào trực tiếp file tương ứng.

  • Search →
    Binary Search

    Tìm kiếm nhị phân trên mảng đã sắp xếp.
  • Sort →
    Bubble Sort

    Sắp xếp nổi bọt bằng hoán đổi phần tử lân cận.
  • Sort →
    Heap Sort

    Sắp xếp dựa trên cấu trúc heap nhị phân.
  • Divide →
    Merge Sort

    Chia để trị và trộn lại theo thứ tự tăng dần.
  • Pivot →
    Quick Sort

    Sắp xếp nhanh với chiến lược chọn pivot.
