Demo thuật toán Bubble Sort

Bubble Sort sắp xếp bằng cách so sánh hai phần tử liền kề. Nếu phần tử bên trái lớn hơn phần tử bên phải, chúng sẽ được đổi chỗ. Sau mỗi lượt, phần tử lớn nhất sẽ “nổi” về cuối mảng.

Nhấn “Chạy tự động” hoặc “Từng bước” để bắt đầu.
function bubbleSort(arr) {
  for (let i = 0; i < arr.length - 1; i++) {
    for (let j = 0; j < arr.length - i - 1; j++) {
      if (arr[j] > arr[j + 1]) {
        let temp = arr[j];
        arr[j] = arr[j + 1];
        arr[j + 1] = temp;
      }
    }
  }
}
